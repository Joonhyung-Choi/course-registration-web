
## 📖 프로젝트 소개
수강신청 웹 개발 프로젝트

## 🔍 기능 설명

기존의 수강신청 매커니즘 변경
- 로그인 및 회원가입
- 수강신청 및 예비수강신청 기능
- 교과목 검색 기능
- 개인별 수강신청 과목 확인 기능
